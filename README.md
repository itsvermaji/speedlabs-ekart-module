<h1>SpeEdLabs Ekart module </h1>

## Installation

1. Go to `cd speedlabs-ekart-app/server`.
2. Type `npm install`
3. Now type `cd ../client`
4. Type `npm install` to install all the <b>react</b> dependencies.
5. Start your Mysql database on the local server.
